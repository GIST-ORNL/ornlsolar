This folder contains shapefiles with the solar panels either manually drawn by hand or detected by ORNL's automatic detection algorithm. The following text describes each 
subfolder in greater detail:

subfolders:

1.) "boston" - contains shapefiles with Boston solar panels that were either manually detected by hand or automatically detected by ORNL's automatic detection algorithm. 

2.) "dc" - contains shapefiles with DC solar panels that were either manually detected by hand or automatically detected by ORNL's automatic detection algorithm. 

subsubfolders:

2.) "panels_aut_city" - contains shapefiles with solar panels that were automatically detected by ORNL's automatic detection algorithm for either city = boston
or city = dc. 

2.) "panels_man_city" - contains shapefiles with solar panels that were manually detected by hand for either city = boston or city = dc.
